# denton-rain-maker

KiCad schematic and PCB for **Denton's Rainmaker** — a desk recirculating water/mist feature.

## Files

- `denton-rain-maker.kicad_pro` — project
- `denton-rain-maker.kicad_sch` — schematic
- `denton-rain-maker.kicad_pcb` — board layout

Open the `.kicad_pro` file in [KiCad](https://www.kicad.org/) 8 or newer.
